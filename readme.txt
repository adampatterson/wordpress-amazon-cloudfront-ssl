Allows WordPress to work properly with Terminated SSL from Amazon CloudFront or CloudFlare.

== Description ==

Using CloudFlare's Flexible SSL on WordPress isn't as simple as just turning it on.

This plugin forms an **integral part** to enabling Flexible SSL on WordPress and prevents infinite redirect loops when loading WordPress sites under CloudFlare's Flexible SSL system.

Based off of [Download
CloudFlare Flexible SSL](https://en-ca.wordpress.org/plugins/cloudflare-flexible-ssl/)

[View on Github](https://github.com/adampatterson/wordpress-amazon-cloudfront-ssl).
